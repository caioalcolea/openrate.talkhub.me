# Ícones do PWA

Adicione dois PNGs aqui antes do go-live (referenciados em `manifest.webmanifest`):

- `icon-192.png` — 192×192
- `icon-512.png` — 512×512 (usável como maskable)

Enquanto não existirem, a instalação ("Adicionar à tela inicial") funciona, mas
sem ícone próprio. Não versionamos binários no repositório de código.
